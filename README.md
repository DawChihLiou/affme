Simple CLI in Rust to show affirmations to someone.

![Demo](./demo.gif)

## Usage

```bash
Usage: cargo run -- --name <NAME>

Options:
  -n, --name <NAME>  Name of the person to affirm
  -h, --help         Print help information
  -V, --version      Print version information
```

## Contributing

To build, run

```bash
cargo build
```

To format, run 

```bash
cargo fmt
```
